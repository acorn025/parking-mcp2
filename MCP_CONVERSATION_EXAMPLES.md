# Play MCP 대화 예시

## 예시 1: 서울 지역 주차장 검색 (실시간 정보 포함)

**사용자:**
서울시청 근처 주차장 찾아줘. 실시간 주차 가능 대수도 알려줘.

**AI:**
서울시청 주변 주차장을 검색하고 실시간 정보를 확인하겠습니다.

[search_nearby_parking 호출: address="서울시 중구 세종대로 110", radius=1000]

**응답 예시:**
서울시청 근처에서 5개의 주차장을 찾았습니다:

1. **시청 공영주차장**
   - 주소: 서울시 중구 세종대로 110
   - 실시간 정보: 15대 주차 가능 / 전체 100대
   - 요금: 기본 10분당 500원
   - 운영시간: 평일 06:00-24:00, 주말 06:00-24:00

2. **덕수궁 주차장**
   - 주소: 서울시 중구 세종대로 99
   - 실시간 정보: 8대 주차 가능 / 전체 50대
   - 요금: 기본 10분당 400원
   - 운영시간: 평일 09:00-22:00

## 예시 2: 경기 지역 주차장 검색 (요금 및 운영시간 포함)

**사용자:**
수원 화성행궁 가려는데 주차장 어디 있나요? 요금 정보도 알려주세요.

**AI:**
수원 화성행궁 근처 주차장을 검색하고 요금 정보를 확인하겠습니다.

[search_nearby_parking 호출: address="경기도 수원시 팔달구 정조로 825", radius=1000]

**응답 예시:**
화성행궁 근처에서 3개의 주차장을 찾았습니다:

1. **화성행궁 공영주차장**
   - 주소: 경기도 수원시 팔달구 정조로 825
   - 전체 주차 대수: 80대
   - 요금: 기본 30분 무료, 이후 10분당 500원
   - 운영시간: 평일 09:00-18:00, 토요일 09:00-18:00, 공휴일 09:00-18:00
   - 결제방법: 현금, 카드

2. **수원역 주차장**
   - 주소: 경기도 수원시 팔달구 덕영대로 924
   - 전체 주차 대수: 120대
   - 요금: 기본 10분당 400원
   - 운영시간: 평일 06:00-24:00

## 예시 3: 기타 지역 주차장 검색 (기본 정보만)

**사용자:**
부산 해운대 해수욕장 가는데 주차장 어디 있나요?

**AI:**
부산 해운대 해수욕장 근처 주차장을 검색하겠습니다.

[search_nearby_parking 호출: address="부산광역시 해운대구 해운대해변로 264", radius=1000]

**응답 예시:**
해운대 해수욕장 근처에서 4개의 주차장을 찾았습니다.

**안내:** 해당 지역은 기본 주차장 정보만 제공됩니다. 실시간 주차 가능 대수는 서울/경기 지역에서만 이용 가능합니다.

1. **해운대 해수욕장 주차장**
   - 주소: 부산광역시 해운대구 해운대해변로 264
   - 전체 주차 대수: 200대

2. **해운대 센텀시티 주차장**
   - 주소: 부산광역시 해운대구 센텀중앙로 97
   - 전체 주차 대수: 150대

3. **해운대역 주차장**
   - 주소: 부산광역시 해운대구 해운대해변로 209
   - 전체 주차 대수: 80대

---

## 사용 팁

- **서울 지역**: 실시간 주차 가능 대수, 요금, 운영시간 모두 제공
- **경기 지역**: 요금, 운영시간 정보 제공 (실시간 대수는 제공되지 않음)
- **기타 지역**: 기본 주차장 정보만 제공 (이름, 주소, 전체 대수)

각 지역별로 제공되는 정보가 다르니 참고하세요!
