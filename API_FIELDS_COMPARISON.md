# 서울/경기 API 필드 비교

## 서울 열린데이터 API 필드

### ✅ 운영 정보
- `OPER_SE_NM`: 운영 구분명 (예: "시간제 운영")
- `PRK_STTS_NM`: 주차 상태명 (예: "주차~20분까지 무료이용 가능(주차장 운영시간 표시)")
- `WD_OPER_BGNG_TM`: 평일 운영 시작 시간 (예: "0000")
- `WD_OPER_END_TM`: 평일 운영 종료 시간 (예: "2400")
- `WE_OPER_BGNG_TM`: 주말 운영 시작 시간
- `WE_OPER_END_TM`: 주말 운영 종료 시간
- `LHLDY_OPER_BGNG_TM`: 공휴일 운영 시작 시간
- `LHLDY_OPER_END_TM`: 공휴일 운영 종료 시간

### ✅ 요금 정보
- `PAY_YN_NM`: 유료 여부 (예: "유료")
- `NGHT_PAY_YN_NM`: 야간 유료 여부 (예: "야간 무료")
- `BSC_PRK_CRG`: 기본 주차 요금 (예: 430.0원)
- `BSC_PRK_HR`: 기본 주차 시간 (예: 5.0분)
- `ADD_PRK_CRG`: 추가 주차 요금 (예: 430.0원)
- `ADD_PRK_HR`: 추가 주차 시간 (예: 5.0분)
- `DAY_MAX_CRG`: 일일 최대 요금 (예: 30900.0원)
- `PRD_AMT`: 기간 요금 (예: 176000원)

### ✅ 실시간 정보
- `TPKCT`: 총 주차 대수
- `NOW_PRK_VHCL_CNT`: 현재 주차 대수
- `NOW_PRK_VHCL_UPDT_TM`: 업데이트 시간

## 경기데이터드림 API

### ❌ 현재 상태
- 서비스를 찾을 수 없음
- 엔드포인트 확인 필요
- 사용자가 언급한 운영정보 및 요금 정보는 API가 정상 작동하면 확인 가능

## 결론

**서울 API에는 운영정보와 요금 정보가 모두 포함되어 있습니다!**

현재 서버 코드에서는 실시간 주차 대수만 사용하고 있지만, 운영정보와 요금 정보도 추가로 활용할 수 있습니다.
